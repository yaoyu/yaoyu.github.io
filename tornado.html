介绍

Tornado是一个python web框架和异步网络库(Tornado is a Python web framework and asynchronous networking library).

Tornado可以分为4个主要组件:

  • 一个web框架(RequestHandler)
  • 客户端和服务器端HTTP实现 Client- and server-side implementions of HTTP (HTTPServer and AsyncHTTPClient)
  • 异步网络库(包括IOLoop和IOStream)An asynchronous networking library including the classes IOLoop and IOStream, which serve as the building blocks for the HTTP components and can also be used to implement other protocols.
  • 协程库(tornado.gen),它让异步调用更加直接
