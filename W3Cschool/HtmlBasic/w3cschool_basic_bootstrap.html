使用Bootstrap设计响应式页面

使用Bootstrap也非常简单,你只需要把下面的链接添加到你需要使用Bootstrap来进行布局的应用的头部：

<link rel="stylesheet" href="//maxcdn.bootstrapcdn.com/bootstrap /3.3.1/css/bootstrap.min.css"/>

通过Bootstrap使图片适配手机显示

其实很简单的，通过Bootstrap，我们只要给图片添加 img-responsive 的class属性,图片的宽度就能自动适配你手机屏幕的宽度啦。

<div class="container-fluid">
<h2 class="red-text">CatPhotoApp</h2>

<p>Click here for <a href="#">cat photos</a>.</p>

使文本居中

<h2 class="red-text text-center">your text</h2>

按钮

<button class="btn">Like</button>

//让按钮充满整个屏幕 btn-block
<button class="btn btn-block">Like</button>

//按钮添加颜色 btn-primary
<button class="btn btn-block btn-primary">Like</button>

//按钮添加颜色 btn-info
<button class="btn btn-block btn-info">Info</button>

//按钮添加颜色 btn-danger
<button class="btn btn-block btn-danger">Delete</button>

使用Bootstrap做页面布局

<div class="row">
    <div class="col-xs-4"><button class="btn btn-block btn-primary">Like</button></div>
    <div class="col-xs-4"><button class="btn btn-block btn-info">Info</button></div>
    <div class="col-xs-4"><button class="btn btn-block btn-danger">Delete</button></div>
</div>

span标签来创建行内元素

还记得我们使用 .btn-block来创建填满整行的按钮吗？你可以使用span 标签来创建一个充满整行的内联元素。

"Top 3 things cats hate" 的写法如下：

<p>Top 3 things cats <span class = "text-danger">hate:</span></p>




row, col-xs-8, col-xs-4


使用<div class="row"> 标签把你的第一张猫咪图片跟h2 标签包含起来,然后创建一个<div class="col-xs-8"> 元素把h2元素包含进去,猫咪图片则放到 <div class="col-xs-4">标签里,这样,h2 标签跟猫咪图片就处于同一行中了。



在按钮上使用Font Awesome图标


Font Awesome是一个方便的图标库。 这些图标都是矢量图形，以.svg文件格式存储。 这些图标可以像字体一样被处理,你可以使用像素设置字体大小一样指定这些图标的大小，并且这些图标可以继承父类HTML标签的字体大小。
